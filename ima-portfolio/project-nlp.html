Overview

Built a two-part NLP system: a BERT-based genre classifier that identifies a song's genre from its lyrics using cosine similarity and linguistic pattern analysis, and an LSTM text generation model with temperature tuning and top-p sampling to produce stylistically consistent lyric outputs in a target genre's voice.

Core Concepts

  • BERT fine-tuning for genre classification — embeddings capture semantic and stylistic differences across genres at the token level
  • Cosine similarity analysis to quantify how closely a generated or real lyric adheres to a genre's stylistic fingerprint
  • LSTM generation with temperature and top-p sampling to control creativity vs. stylistic fidelity
  • Large-scale text preprocessing — cleaning, tokenizing, and structuring 10,000+ lyric records with Pandas

Development Notes

Temperature tuning controls the randomness of the LSTM output — lower temperatures produce more genre-faithful text, higher values introduce creative variation:

def sample_with_temperature(logits, temperature=0.8, top_p=0.9):
    logits = logits / temperature
    probs = torch.softmax(logits, dim=-1)
    # Top-p (nucleus) sampling
    sorted_probs, sorted_idx = torch.sort(probs, descending=True)
    cumulative = torch.cumsum(sorted_probs, dim=-1)
    sorted_probs[cumulative > top_p] = 0
    return torch.multinomial(sorted_probs, num_samples=1)
"Every genre has a fingerprint. NLP makes that fingerprint measurable."

The classifier achieved strong separation between genre clusters when visualized with t-SNE on the BERT embeddings — hip-hop and country formed particularly tight, distinct clusters, while pop showed the most overlap with adjacent genres.
