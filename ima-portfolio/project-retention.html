Overview

Built a Random Forest classification model using PySpark on AWS EMR to predict customer churn across a dataset of 7,000+ records. The model surfaces the highest-risk customers and quantifies the revenue exposure associated with predicted churn — enabling proactive retention intervention.

Core Concepts

  • Predictive risk segmentation using Random Forest with hyperparameter tuning via cross-validation
  • Distributed processing on AWS EMR to handle large-scale data transformation with PySpark
  • Revenue impact modeling — each at-risk segment mapped to CLV to produce a dollar figure
  • Feature engineering from behavioral, transactional, and temporal signals
