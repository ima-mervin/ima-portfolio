Overview

Built an ensemble deep learning classifier that combines a Vision Transformer (ViT) and EfficientNet to detect AI-generated or manipulated images. The model is deployed as a live web interface on Hugging Face Spaces, allowing anyone to upload an image and receive an authenticity verdict with visual explainability output.

Core Concepts

  • Ensemble architecture — ViT captures global attention patterns while EfficientNet handles local texture artifacts typical of GAN-generated faces
  • Visual explainability via Grad-CAM heatmaps, highlighting the image regions that triggered the deepfake classification
  • Transfer learning from pretrained ImageNet weights, fine-tuned on a labeled real/fake image dataset
  • Live inference deployed via Gradio on Hugging Face Spaces with a real-time web UI

Development Notes

The ensemble combines soft-voting across both models. Grad-CAM is applied post-inference to produce the visual explanation overlay:

# Ensemble soft voting
vit_prob = vit_model(image)
eff_prob = eff_model(image)
final_pred = (vit_prob + eff_prob) / 2

# Grad-CAM for explainability
cam = GradCAM(model=vit_model, target_layer="blocks[-1]")
heatmap = cam(input_tensor=image)
"In an era of synthetic media, the ability to verify what's real is infrastructure."

The live interface on Hugging Face lets users drag and drop any image to receive a real/fake probability score alongside a heatmap showing which facial regions the model flagged as manipulated.
